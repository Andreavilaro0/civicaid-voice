# Design — Clara / CivicAid Voice

Carpeta central de todo el trabajo de diseno, marca, webapp, videos y marketing de Clara.

**Owner:** Andrea Avila
**Proyecto:** OdiseIA4Good Hackathon — UDIT Feb 2026

---

## Estructura

```
design/
├── branding/              # Identidad visual de Clara
│   ├── logo/              # Versiones del logo (SVG, PNG, variantes)
│   ├── colores/           # Paleta de colores, tokens, fondos
│   ├── tipografia/        # Fuentes seleccionadas, especimenes
│   └── guia-de-marca/     # Brand book, reglas de uso, tono visual
│
├── webapp/                # Pagina web / webapp de Clara
│   ├── mockups/           # Wireframes y mockups de pantallas
│   ├── componentes/       # Componentes UI individuales
│   ├── assets/            # Imagenes y recursos para la web
│   └── paginas/           # Disenos finales por pagina (home, about, demo)
│
├── videos/                # Todo el contenido audiovisual
│   ├── demo/              # Videos de demo del producto
│   ├── promo/             # Videos promocionales / trailer
│   ├── tutoriales/        # Videos explicativos de uso
│   └── assets/            # Clips, musica, efectos, thumbnails
│
├── marketing/             # Materiales de comunicacion
│   ├── redes-sociales/    # Posts, stories, banners para RRSS
│   ├── copy/              # Textos publicitarios, taglines, bios
│   └── materiales-impresos/ # Tarjetas, posters, flyers, QR
│
├── presentaciones/        # Slides y materiales de pitch
│   ├── slides/            # Archivos de presentacion (PPT, Figma, PDF)
│   ├── portadas/          # Portadas 16:9, covers
│   └── recursos/          # Graficos auxiliares para slides
│
└── assets/                # Recursos graficos compartidos
    ├── iconos/            # Set de iconos del proyecto
    ├── ilustraciones/     # Ilustraciones personalizadas
    ├── fotos/             # Fotografias de stock o propias
    └── audio/             # Jingles, efectos de sonido, voiceover
```

---

## Convenciones

- **Nombres de archivo:** minusculas, guiones en vez de espacios (`logo-principal.svg`, no `Logo Principal.svg`)
- **Formatos preferidos:** SVG para vectores, PNG para rasters, MP4 para video, MP3/OGG para audio
- **Versionado:** Si hay multiples versiones, usar sufijo `_v1`, `_v2` (ej: `logo-principal_v2.svg`)
